# Download

:::caution Not Yet Available
Airwaves OS is under active development and is not yet available for public download. Join the [Airframes Discord](https://discord.gg/airframes) to follow development progress and be notified when images are available.
:::

## In the Meantime

While AROS is in development, you can set up a feeder station using:

- **[Airframes Installers](/docs/feeding/how)** — One-command setup scripts for common decoder configurations
- **[Docker / ACARS Hub](/docs/feeding/Docker)** — Container-based setup with a web dashboard
- **[Manual installation](/docs/decoders/clients)** — Build and configure individual decoders yourself
