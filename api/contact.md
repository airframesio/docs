# Contact

## API Access

If you are interested in early access to the API, please contact [api@airframes.io](mailto:api@airframes.io) with:

- Your intended use case
- Whether you are a current Airframes feeder (and your station ID if so)
- Expected request volume

## General Support

For general questions about Airframes, join the [Discord](https://discord.gg/airframes) or post on the [Community Forum](https://community.airframes.io).

For technical issues with your feeder setup, see the [Troubleshooting](/docs/troubleshooting) guide.
